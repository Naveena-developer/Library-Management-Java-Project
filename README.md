# 📚 Library Management System (Java)

## 🔹 Project Description
This is a console-based Library Management System developed using Java.  
It helps manage book records and perform operations like adding, searching, issuing, and returning books.

---

## 🚀 Features
- Add new books with unique ID
- View all books
- Search book by name 🔍
- Issue book
- Return book
- Prevent duplicate IDs ❌
- Display "Book not found" message

---

## 🛠️ Technologies Used
- Java
- OOP (Object-Oriented Programming)
- ArrayList

---

## ▶️ How to Run

1. Compile the program:
   javac LibrarySystem.java

2. Run the program:
   java LibrarySystem
